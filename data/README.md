# Data

This analysis uses the GLODAP v2.2023 dataset.

## Download Instructions

1. Visit the GLODAP website: https://www.glodap.info/
2. Download the **Merged Master File** (CSV format)
3. Place the file `GLODAPv2.2023_Merged_Master_File.csv` in this directory

## Reference

Lauvset, S. K., et al. (2024). GLODAPv2.2023: the third update of GLODAPv2.
Earth System Science Data, 16, 2047-2072. https://doi.org/10.5194/essd-16-2047-2024
